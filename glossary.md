# Glossary

## LLM
Large Language Model trained to predict tokens.

## RAG
Retrieval-Augmented Generation. Combines vector retrieval with generation.

## Agent
An LLM system that can reason, use tools, and iterate toward a goal.

## Tool
A function exposed to the model that it can call during reasoning.

## MCP
Model Context Protocol. A standardized way for models to access tools and data.

## Embeddings
Vector representations of text used for similarity search.
